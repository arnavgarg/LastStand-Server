# LastStand-Server

Server for Last Stand Client (https://github.com/arnavgarg/LastStand-Client)

**Usage**
1. `cd /path/to/LastStand-Server`
2. `export GOPATH=/path/to/LastStand-Server`
3. `cd src/main`
4. `go get`
5. `cd ../..`
6. `go install main`
7. `./bin/main`

**Dependencies**
- Mux Router (https://github.com/gorilla/mux)
